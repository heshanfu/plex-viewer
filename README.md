
# plex-viewer

A simple electron app for viewing plex and with support for keyboard shortcuts.

![screenshot_1](https://imgur.com/svVqU7o.jpg)

### Shortcuts
* Toggle minimize - `Shift + Ctrl + X`
* Toggle window frame (chrome) - `Shift + Ctrl + Z`
