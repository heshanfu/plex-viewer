
# electron-create-react-app-boilerplate

A simple electron app boilerplate powered by create-react-app. Supports hot reloading locally.

## Requirements
 * Node >= 8.2.0

## Getting started
1. `npm install`
2. `npm start`
3. ???
4. Profit

For hot reloading `SET NODE_ENV=development`
